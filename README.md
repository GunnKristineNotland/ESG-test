# ESG Senior Advisor — enkel hjemmeside

Dette er et enkelt nettsted for å presentere konsulenttjenester innen ESG (bærekraft, styring og rapportering).

Innhold:
- `index.html` — landingsside
- `css/style.css` — enkel styling

For å publisere på GitHub Pages:
1. Push filene til `main`-branch (eller `gh-pages` hvis du foretrekker det).
2. Gå til repository > Settings > Pages og velg publiseringskilde (f.eks. `main` / `/(root)`).
3. Vent et par minutter — nettstedet blir publisert på `https://<brukernavn>.github.io/<repo>`.

Enkle git-kommandoer for å komme i gang:
```bash
git clone https://github.com/GunnKristineNotland/ESG-test.git
cd ESG-test
# legg til filene fra denne malen
git add .
git commit -m "Legg til enkel ESG landingsside"
git push origin main
```
